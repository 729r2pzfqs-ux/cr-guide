Referencia de Pictogramas GHS

El Sistema Globalmente Armonizado (GHS) utiliza estos pictogramas para identificar peligros:

GHS01 - Explosivo
Explosivos, sustancias autorreactivas
GHS02 - Inflamable
Gases, líquidos, sólidos inflamables
GHS03 - Comburente
Puede provocar o intensificar un incendio
GHS04 - Gas a Presión
Gases bajo presión
GHS05 - Corrosivo
Corrosión cutánea, lesiones oculares
GHS06 - Tóxico
Toxicidad aguda (mortal o tóxico)
GHS07 - Irritante
Irritante, sensibilizante, narcótico
GHS08 - Peligro para la Salud
Carcinógeno, mutágeno, daño a órganos
GHS09 - Peligro Ambiental
Peligroso para el medio acuático

¿Qué es una Ficha de Datos de Seguridad (FDS)?

Una Ficha de Datos de Seguridad es un documento que proporciona información completa sobre una sustancia química o mezcla. Es obligatoria para productos químicos peligrosos y contiene 16 secciones estandarizadas:

1. Identificación de la sustancia
2. Identificación de peligros
3. Composición/ingredientes
4. Primeros auxilios
5. Medidas de extinción
6. Vertido accidental
7. Manipulación y almacenamiento
8. Control de exposición/EPP
9. Propiedades físicas/químicas
10. Estabilidad y reactividad
11. Información toxicológica
12. Información ecológica
13. Eliminación
14. Información de transporte
15. Información reglamentaria
16. Otra información

⚠️ Esta herramienta proporciona solo resúmenes. Consulte siempre la Ficha de Datos de Seguridad completa para información detallada.
