Résistance Chimique du NBR

Le NBR (caoutchouc nitrile-butadiène) est la norme industrielle pour la résistance aux huiles et carburants. Également connu sous le nom de Buna-N, c'est l'élastomère le plus utilisé pour les joints hydrauliques, les joints toriques et les composants de systèmes de carburant.

Applications courantes

Systèmes de carburant

Durites carburant, joints de réservoir, pompes essence, systèmes diesel

Hydraulique

Joints hydrauliques, joints toriques, joints de vérin, pompes

Automobile

Joints moteur, joints de transmission, joints de carter d'huile

⚠️ Non recommandé pour

Le NBR se dégrade au contact de :

  • Exposition ozone et UV (usage extérieur)
  • Cétones (acétone, MEK)
  • Esters et éthers
  • Acides forts
  • Hydrocarbures chlorés
  • Liquide de frein (à base de glycol)

Conseil : Pour les applications extérieures ou l'exposition à l'ozone, utilisez l'EPDM. Pour les produits chimiques extrêmes, considérez le Viton® (FKM).
